# Instruções para Adicionar a Foto

## Para que a foto apareça como fundo do progresso da jornada:

1. Salve a foto que você enviou como: `amor-jornada.jpg`
2. Coloque o arquivo nesta pasta: `public/images/`
3. O caminho final deve ser: `public/images/amor-jornada.jpg`

A foto será exibida de forma sutil (15% de opacidade) no fundo do painel "Seus 90 Dias", simbolizando o amor que dá força para a jornada.

## Formato aceito:
- Nome: `amor-jornada.jpg`
- Formato: JPG, JPEG, PNG
- Resolução recomendada: 1920x1080 ou superior para melhor qualidade
